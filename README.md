# kodutoo-1

**Autor:** Elisabet Peterson

## Kirjeldus
See on veebipõhine digitaalne kell, mis on loodud täitma ekraanisäästja või lauakella rolli. Kell on disainitud kasutades protsendilisi väärtusi (vw), et see sobituks igale ekraanile.

### Funktsionaalsus:
1. **Fondi suurus:** Saab suurendada ja vähendada (+/- nupud).
2. **Taustavärv:** Muudetav suvaliseks värviks.
3. **Tekstivärv:** Kasutaja saab muuta numbrite ja teksti värvi.
4. **Ajavorming:** Valik 24h ja 12h režiimide vahel.
5. **Kirjatüüp:** Valik erinevate fondistiilide vahel.
6. **Keelevalik:** Kell tuvastab eesti keele ja kuvab nädalapäevad/kuud vastavalt.

### Kasutatud tehnoloogiad
- HTML5, CSS3, JavaScript 
- Kasutatud on objektorienteeritud lähenemist 
- Koodi loomisel on kasutatud Gemini tehisintellekti abi (struktuur ja algne idee mille põhjalt edasi minna).
